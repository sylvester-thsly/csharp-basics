### Basic Method Structure
```csharp
void SayHello()
{
    Console.WriteLine("Hello!");
}
```

- `void` = Returns nothing
- `SayHello` = Method name
- `()` = Parameters (empty for now)
- `{ }` = Method body (the code)

### Calling a Method
```csharp
SayHello();  // This runs the method
```
