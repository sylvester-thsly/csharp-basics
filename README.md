## 📖 About This Repository

This repository documents my journey learning C# from the ground up. Each "Tag" (day/module) covers a specific topic with:

- 📝 **Detailed Notes** (`Notizen.md`) - Theory and concepts explained
- 💻 **Practice Projects** - Hands-on code to reinforce learning
- ✅ **Quiz Solutions** - Self-assessment and problem-solving
- 📋 **Cheat Sheets** (`Merksatz.md`) - Quick reference summaries

**Learning Philosophy:** Write every line of code myself. No copy-paste, no shortcuts. True understanding comes from practice.

## 📊 Learning Methodology

I follow a **30% Theory / 70% Practice** approach:
1. **Read** the concept (15 min)
2. **Explain** it in my own words (5 min)
3. **Code** 3 examples (15 min)
4. **Review** from memory the next day (5 min)
